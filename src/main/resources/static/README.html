本地开发

git clone项目到本地，修改配置文件[resources/application*.*]中的数据库连接信息，导入SQl/*.SQL文件生成所有表

安装部署

可以使用docker方式部署，也可支持-jar方式，服务器可以用阿里云

项目设计

功能需求

主页

  • 博客汇总，以列表形式展示文章，并附上文章作者、发布日期、分类情况以及文章简要

  • 点击文章标题进入文章详情页

  • 能够以分类形式查看文章

  • 个人介绍、联系方式

  • 置顶文章

后台管理

  • 发布文章（用户权限1） 1.使用markdown编辑器，支持插入代码，插入图片等功能

  • 修改文章（用户权限2） 1.分页展示文章信息 2.改变文章基本信息 3.可对文章进行编辑 4.可以删除文章

  • 用户管理（用户权限3） 1.可以添加权限1-3用户 2.可以删除权限1-2用户 3.可以修改权限1-2用户信息 4.可以修改用户自身信息
